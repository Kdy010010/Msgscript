Welcome to MsgScript Introduction

MsgScript is a simple scripting language that allows you to execute various actions based on commands. The commands follow the format: action:message.

Supported Actions:

  • print:message - Print the message to the console.
  • console.print:message - Print the message to the console with a prefix.
  • innerhtml:message - Append the message to the body of the document.
  • alert:message - Show an alert with the given message.
